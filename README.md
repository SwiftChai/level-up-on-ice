# Level Up On Ice

A responsive website for figure skating seminars featuring coaching from elite skaters and choreographers.

**Note:** This is a demo project for web development practice. This is not a real seminar and no information submitted through the form is stored.

## About

Level Up On Ice is a fictional figure skating seminar website created to practice web development and design skills. The site may be updated periodically as I experiment with new techniques and design concepts.

## Features

- **Responsive Design**: Optimized for desktop, tablet, and mobile devices
- **Registration Form**: Interactive sign-up interface (no data stored)
- **Clean Layout**: Professional presentation with smooth navigation

## Technologies Used

- HTML5
- CSS3
- Responsive Design (Media Queries)
